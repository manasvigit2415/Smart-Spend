Problem Statement & Scope

1. Problem Statement

Managing daily finances is often neglected due to complex banking apps or lack of discipline. This project solves that by providing a fast, offline tool to record transactions instantly.

2. Scope

SmartSpend Console allows a single user to log in securely, record financial data, and view summaries. Data is stored locally in text files for privacy.

3. Target Users

Students managing allowances.

Freelancers tracking gig income.

4. High-Level Features

Auth: Secure Login/Register.

Ledger: Add/View transactions.

Report: Calculate Net Balance.
